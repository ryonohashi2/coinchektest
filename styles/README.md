# Styles

スタイル関連ファイル

## 構成
- `globals.css` - グローバルスタイル
- `components.css` - コンポーネント固有スタイル
- `utilities.css` - ユーティリティクラス

## 設計方針
- Tailwind CSS をベースとする
- カスタムCSSは最小限に抑制
- レスポンシブデザイン対応
- ダークモード対応（将来的に）

## カラーパレット
- Primary: Blue系
- Secondary: Gray系
- Success: Green系
- Warning: Yellow系
- Danger: Red系
